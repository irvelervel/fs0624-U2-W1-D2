POSITION: STATIC

Il valore "static" per la proprietà position è quello di default. Un elemento dotato di position: static NON è posizionato all'interno della pagina in nessun modo particolare. Si dichiara esplicitamente in casi particolari, magari all'interno di una media query per riassegnarne il valore di default in casi particolari, oppure quando non si desideri che un figlio erediti il comportamento del padre.
